# Contributing to Study Schedule Generator

Thank you for considering contributing to the **Study Schedule Generator** project! We welcome contributions from the community to make this project better.

## How to Contribute

### Reporting Issues
If you find a bug, have a suggestion, or experience issues with the project, please [open an issue](https://github.com/yourusername/Study-Schedule-Generator/issues). Be sure to include as much detail as possible, including steps to reproduce the issue, and screenshots if applicable.

### Feature Requests
We are always open to new ideas! If you have a feature suggestion, [open an issue](https://github.com/yourusername/Study-Schedule-Generator/issues) and label it as a feature request. Please provide as much context as possible.

### Submitting a Pull Request
If you'd like to contribute code:
1. **Fork** the repository.
2. Create a new branch:
   ```bash
   git checkout -b feature-branch
